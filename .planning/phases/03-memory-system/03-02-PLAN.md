Create Ebbinghaus-inspired decay calculations:

1. Define DECAY_STRENGTH constants (hours for half-life):
   ```typescript
   export const DECAY_STRENGTH: Record<MemoryCategory, number> = {
     persistent: 168,   // ~1 week - facts, relationships, preferences
     temporal: 24,      // ~1 day - recent events, current emotions
     contextual: 4,     // ~4 hours - conversation-specific context
   };
   ```

2. Create calculateDecay function:
   - Takes: memory (Memory), now (timestamp)
   - Calculates hours since lastAccessed
   - Gets base strength from DECAY_STRENGTH by category
   - Boosts strength based on accessCount: `strength * (1 + Math.log10(accessCount + 1))`
   - Returns decay factor using Ebbinghaus formula: `Math.exp(-hoursSinceAccess / boostedStrength)`
   - Result is 0-1 where 1 = no decay, 0 = fully decayed

3. Create calculateRelevanceScore function:
   - Takes: memory (Memory), now (timestamp)
   - Calculates decay factor
   - Normalizes importance to 0-1: `importance / 10`
   - Returns: `normalizedImportance * decay`
   - Score ranges 0-1, used for ranking memories

4. Create calculateKeywordMatch function (helper for retrieval):
   - Takes: memoryContent (string), contextText (string)
   - Extracts words >3 characters from both
   - Counts matches (case-insensitive)
   - Returns: `Math.min(matches / 5, 1)` (capped at 1.0)

Import Memory and MemoryCategory from '../types/memory'.
