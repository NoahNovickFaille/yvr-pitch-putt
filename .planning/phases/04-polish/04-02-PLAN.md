**src/stores/chatStore.ts:**
Add clearConversation method to interface and implementation:
```typescript
clearConversation: () => {
  storage.remove('chat_messages');
  storage.remove('conversation_meta');
  set({ messages: [], conversationMeta: null });
}
```

**src/stores/memoryStore.ts:**
Add clearAll method to interface and implementation:
```typescript
clearAll: () => {
  storage.remove('memories');
  set({ memories: [] });
}
```

IMPORTANT: Do NOT use storage.clearAll() - this would remove onboarding state. Only remove specific keys per research "Pitfall 3: MMKV Storage Key Conflicts".
