<task type="auto">
  <name>Task 2: Create EmbeddingStorage module</name>
  <files>src/services/embedding/EmbeddingStorage.ts</files>
  <action>
Create `src/services/embedding/EmbeddingStorage.ts` with:

1. Import dependencies:
   - storage from storage/storage
   - EmbeddingVector from types/embedding

2. Define EMBEDDING_PREFIX constant: 'emb:'

3. Implement `storeEmbedding(memoryId: string, embedding: EmbeddingVector): void`:
   - Convert number[] to Float32Array
   - Convert Float32Array to Uint8Array (via shared ArrayBuffer)
   - Store using storage.set(`${EMBEDDING_PREFIX}${memoryId}`, uint8.buffer)
   - Log in __DEV__ mode

4. Implement `getEmbedding(memoryId: string): EmbeddingVector | null`:
   - Call storage.getBuffer(`${EMBEDDING_PREFIX}${memoryId}`)
   - If null/undefined, return null
   - Convert buffer to Float32Array
   - Return Array.from(float32) to get number[]
   - Handle potential errors gracefully (corrupted data)

5. Implement `hasEmbedding(memoryId: string): boolean`:
   - Return storage.contains(`${EMBEDDING_PREFIX}${memoryId}`)

6. Implement `deleteEmbedding(memoryId: string): void`:
   - Call storage.delete(`${EMBEDDING_PREFIX}${memoryId}`)

7. Implement `getAllEmbeddingKeys(): string[]`:
   - Get all keys from storage
   - Filter keys starting with EMBEDDING_PREFIX
   - Return array of memory IDs (strip prefix)
   - Useful for migration progress tracking

8. Export all functions

Storage format note: 256 floats * 4 bytes = 1024 bytes per embedding. MMKV handles this efficiently with single-copy operations.
  </action>
