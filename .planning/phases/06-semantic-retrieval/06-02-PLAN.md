  <action>
Create SemanticRetrieval.ts with the following functions:

**1. scoreMemory() function:**
```typescript
function scoreMemory(
  memory: Memory,
  queryEmbedding: EmbeddingVector,
  now: number = Date.now()
): ScoredMemory | null
```
- Get memory embedding via getEmbedding(memory.id)
- Return null if no embedding (skip gracefully)
- Calculate semanticScore via cosineSimilarityNormalized()
- Calculate decayScore via calculateDecay()
- Calculate importanceScore as memory.importance / 10
- Calculate finalScore using RETRIEVAL_WEIGHTS
- Return ScoredMemory object

**2. isIdentityMemory() helper:**
```typescript
function isIdentityMemory(memory: Memory): boolean
```
- Return true for type 'person'
- Return true for type 'fact' if content matches identity patterns:
  - Contains "name is" (user's name)
  - Contains "works as" or "job is" (occupation)
  - Contains "lives in" (location)
  - Contains "is a " (identity statement)
- Return false otherwise

**3. retrieveByKeywords() fallback:**
```typescript
function retrieveByKeywords(
  query: string,
  memories: Memory[],
  count: number
): Memory[]
```
- Use calculateKeywordMatch() from MemoryDecay
- Score all memories, sort descending, return top N
- This is the fallback when EmbeddingService is not ready

**4. retrieveMemories() main function:**
```typescript
async function retrieveMemories(
  query: string,
  allMemories: Memory[],
  config: RetrievalConfig = DEFAULT_RETRIEVAL_CONFIG
): Promise<Memory[]>
```

Implementation:
1. Check EmbeddingService.isReady() - if not, call retrieveByKeywords() with count = config.maxIdentity + config.maxTopicRelevant + config.maxRecent
2. Generate query embedding ONCE: await EmbeddingService.embed(query)
3. Separate identity memories using isIdentityMemory(), take up to config.maxIdentity
4. Score remaining memories using scoreMemory()
5. Filter by config.semanticThreshold, sort by finalScore
6. Take top config.maxTopicRelevant as topicRelevant
7. Get recent memories (not already included), sorted by lastAccessed, take config.maxRecent
8. Combine: [...identity, ...topicRelevant, ...recent]
9. Log performance in __DEV__: duration and counts

Add JSDoc comments referencing requirements (SEM-01 through SEM-05).

Imports needed:
- Memory from '../../types/memory'
- ScoredMemory, RetrievalConfig, RetrievalResult from '../../types/retrieval'
- RETRIEVAL_WEIGHTS, DEFAULT_RETRIEVAL_CONFIG from '../../constants/retrieval'
- EmbeddingService from '../embedding/EmbeddingService'
- getEmbedding from '../embedding/EmbeddingStorage'
- cosineSimilarityNormalized from '../embedding/CosineSimilarity'
- calculateDecay, calculateKeywordMatch from './MemoryDecay'
- EmbeddingVector from '../../types/embedding'
