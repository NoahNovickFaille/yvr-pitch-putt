# Expo Template Constants

Theme colors and design tokens from the Expo template.

## Files

- **theme.ts** - Light and dark color palettes with semantic names (text, background, tint, icon, tabIconDefault, tabIconSelected)

## Color System

Two complete palettes:
- `Colors.light` - Light mode colors
- `Colors.dark` - Dark mode colors

Access via `useThemeColor()` hook which automatically switches based on system appearance.

## App-Specific Constants

For app-specific configuration (model URLs, decay rates, etc.), see `src/constants/`.
